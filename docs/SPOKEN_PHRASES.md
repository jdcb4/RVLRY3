# Spoken Phrases

These are the spoken prompts worth recording later to complement the non-verbal audio cues already in the app.

| Phrase | Where it would be used | Game(s) | Priority |
| --- | --- | --- | --- |
| Turn starts now | Immediately when a timed turn begins | WhoWhatWhere, HatGame | Must-have |
| Five seconds left | When a timed turn reaches 5 seconds remaining | WhoWhatWhere, HatGame | Must-have |
| Time | When a timed turn expires | WhoWhatWhere, HatGame | Must-have |
| Pass to the next player | Before a hidden handoff or reveal screen | Imposter, DrawNGuess, local HatGame | Must-have |
| Voting starts now | When the Imposter clue round ends and voting opens | Imposter | Must-have |
| Results revealed | When a round or match result screen appears | Imposter, WhoWhatWhere, DrawNGuess, HatGame | Nice-to-have |
| Phase one: Describe | At the start of HatGame phase 1 | HatGame | Must-have |
| Phase two: One word | At the start of HatGame phase 2 | HatGame | Must-have |
| Phase three: Charades | At the start of HatGame phase 3 | HatGame | Must-have |
| Phase complete | When HatGame advances between phases | HatGame | Must-have |
| Your team is up | On between-turn screens before the active describer starts | WhoWhatWhere, HatGame | Nice-to-have |
| Start when ready | On describer handoff screens before a timed turn | WhoWhatWhere, HatGame | Nice-to-have |
| Reveal your role | On hidden role handoff screens | Imposter | Nice-to-have |
| Submit your clue | On each Imposter clue turn | Imposter | Nice-to-have |
| Draw the prompt | When a DrawNGuess drawing step is revealed | DrawNGuess | Nice-to-have |
| Guess the drawing | When a DrawNGuess guessing step is revealed | DrawNGuess | Nice-to-have |
| Chain complete | When the DrawNGuess reveal chain is ready | DrawNGuess | Nice-to-have |
| Save your clue pack | In the HatGame lobby after editing personal clues | HatGame | Nice-to-have |
| Everyone has submitted | When all HatGame clue packs are ready in the lobby | HatGame | Nice-to-have |
| Return to the lobby for another round | On final result screens | All games | Nice-to-have |

Templated phrases worth considering later with TTS or stitched recordings:

- `Pass to {playerName}` for all handoff states
- `{teamName} are up next` for between-turn team games
- `{describerName} is describing` for live team turns
