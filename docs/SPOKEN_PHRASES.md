# Spoken Phrases

These are the spoken prompts that still make sense to record later. The app already uses non-verbal sound cues today.

## Core system phrases

| Phrase | Where it would be used | Game(s) | Priority |
| --- | --- | --- | --- |
| Turn starts now | When a timed team turn begins | WhoWhatWhere, HatGame | Must-have |
| Five seconds left | At the five-second warning | WhoWhatWhere, HatGame | Must-have |
| Time | When the turn expires | WhoWhatWhere, HatGame | Must-have |
| Results revealed | When a round or match result screen appears | All | Nice-to-have |
| Pass to the next player | Before a hidden handoff or private setup screen | Local modes | Must-have |

## Imposter

| Phrase | Where it would be used | Priority |
| --- | --- | --- |
| Reveal your role | Before each secret role reveal | Nice-to-have |
| Your turn. Say a word. | At the start of each spoken clue turn | Must-have |
| Discuss as a group | When spoken rounds end | Must-have |
| Voting starts now | When the host opens voting | Must-have |
| Lock in your vote | On private voting screens | Nice-to-have |

## WhoWhatWhere

| Phrase | Where it would be used | Priority |
| --- | --- | --- |
| Your team is up | On between-turn screens | Nice-to-have |
| Start when ready | On describer handoff screens | Nice-to-have |
| Return to a skipped word | When skipped words are waiting and the user can bring one back | Nice-to-have |

## DrawNGuess

| Phrase | Where it would be used | Priority |
| --- | --- | --- |
| Draw now | Start of a draw round | Must-have |
| Guess now | Start of a guess round | Must-have |
| Pass the book on | Dummy/pass round in odd-player online games and local handoffs | Must-have |

## HatGame

| Phrase | Where it would be used | Priority |
| --- | --- | --- |
| Save your clue pack | During private clue-entry setup | Nice-to-have |
| Everyone has submitted | When all clue packs are ready | Nice-to-have |
| Phase one: Describe | At HatGame phase 1 start | Must-have |
| Phase two: One word | At HatGame phase 2 start | Must-have |
| Phase three: Charades | At HatGame phase 3 start | Must-have |
| Phase complete | When HatGame rolls to the next phase | Must-have |
